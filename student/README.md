# Java Kurs – Aufgabenprojekt (Student Edition)

Willkommen zum Java Kurs Projekt!

Dieses Projekt enthält:

* **Stub-Methoden**, die du selbst implementieren sollst
* **fertige Unit-Tests (JUnit 5)**, die deine Lösungen überprüfen
* **Java 25** als Zielplattform

Deine Aufgabe ist es, **alle TODOs zu implementieren**, bis die Tests erfolgreich durchlaufen (**grün**).

---

## Voraussetzungen

* Java 25
* Maven 3.9+
* IntelliJ IDEA oder VS Code empfohlen

---

## Projekt bauen und Tests ausführen

### Tests starten

```
mvn test
```

### Komplett builden

```
mvn clean install
```

---

## Aufgaben

Öffne die Klassen unter:

```
src/main/java/
```

Dort findest du Methoden wie:

```java
public int sumPositive(List<Integer> values) {
    // TODO: implementieren
    throw new UnsupportedOperationException("TODO");
}
```

Die Tests definieren genau, was diese Methoden tun müssen.

---

## Ziel

Alle Tests müssen erfolgreich sein.
Wenn ein Test fehlschlägt, lies die Fehlermeldung – sie hilft dir, die Implementierung anzupassen.

Viel Erfolg!
